# LINE スケジュール通知くん

スケジュールを管理し、予定が近づくと LINE に通知するアプリです。
操作は Web アプリで行い、LINE は通知専用です。

## できること

### Web アプリ

- **トップ**: 日程未定＋未来のスケジュール一覧。イベント名・集合時間・備考をその場で編集（入力欄から外れると保存）、最下行から新規追加、行ごとに削除。イベント名から詳細ページへ
- **詳細**: スケジュールのタイトル・集合時間・備考の表示と、参加者の追加・削除
- **過去**: 過去のスケジュール一覧（詳細ページへ遷移可）

### LINE 通知

- 予定の **2 週間前・1 週間前・3 日前・1 日前**にグループへリマインド通知
- 「**BOT**」（大文字小文字問わず）を含むメッセージを送ると、Web アプリの URL を返信

## Getting Started

### 1. インストール

```bash
npm install
```

### 2. ローカルで UI を確認する

```bash
npm run dev
```

GAS 外ではダミーデータで動くため、画面の見た目・操作を単体で確認できます。

### 3. GAS へデプロイする

```bash
npm run push   # ビルドして clasp push
```

その後 GAS で**ウェブアプリとしてデプロイ**します。

### 4. 必要な設定

**スクリプトプロパティ**

| キー | 用途 |
| --- | --- |
| `CHANNEL_ACCESS_TOKEN` | LINE Messaging API のチャネルアクセストークン |
| `GROUP_ID` | 通知・返信先のグループ ID |
| `WEB_APP_URL` | デプロイで発行された Web アプリの URL |

**スプレッドシート**（2 シート）

- スケジュール一覧: `id` / `イベント名` / `集合時間` / `備考`
- 参加者: `イベントid` / `参加者名`

**トリガー**

- `notify` を時間主導型トリガーに登録すると、リマインド通知が定期実行されます。
